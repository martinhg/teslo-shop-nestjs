# Teslo API

1. Clonar el repositorio

2. Ejecutar el comando
```
npm install
```

3. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar el proyecto en el modo desarrollo
```
npm run start:dev
```

7. Ejecutar SEED
```
http://localhost:3000/api/seed
```
